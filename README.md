<h3> How to Play </h3>
<ul> 
  <li> To moves the spacecraft you can use ⬅️⬆️➡️⬇️ </li>
  <li> to boost the spacecraft press the ⬆️ upward arrow key </li>
  <li> Every asteroid you dodge you get a point. </li>
</ul>
